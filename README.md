# Java Console Calculator

A lightweight, command-line-based calculator written in Java. This application performs basic arithmetic operations and provides a clean, interactive user interface directly in the terminal.

---

## Features

* **Basic Arithmetic:** Supports Addition, Subtraction, Multiplication, and Division.
* **Input Validation:** Handles edge cases like division by zero and invalid user inputs gracefully.
* **Interactive Loop:** Allows users to perform multiple calculations in a single session without restarting the app.
* **Clean Console UI:** Easy-to-read menus and formatted output.

---

## Technologies Used

* **Language:** Java (JDK 8 or higher recommended)
* **APIs:** `java.util.Scanner` for handling user input

---

## Getting Started

### Prerequisites

Ensure you have the Java Development Kit (JDK) installed on your machine. You can check your version by running:

```bash
java -version
Installation & Running
Clone the repository (or download the source files):

Bash
git clone [https://github.com/yourusername/java-console-calculator.git](https://github.com/yourusername/java-console-calculator.git)
cd java-console-calculator
Compile the Java file:

Bash
javac Calculator.java
Run the application:

Bash
java Calculator
How to Use
Launch the application.

Select an operation from the menu by entering the corresponding number (1-4).

Enter the first number, then the second number when prompted.

View the result.

Choose whether to perform another calculation (Y/N).

Example Usage
Plaintext
=== Java Console Calculator ===
1. Add (+)
2. Subtract (-)
3. Multiply (*)
4. Divide (/)
5. Exit
==============================
Enter your choice (1-5): 1

Enter first number: 12.5
Enter second number: 7.5

Result: 12.5 + 7.5 = 20.0
